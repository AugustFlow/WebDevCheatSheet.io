HTML Table Reference

Table Tags

Tag Name Description
<table> Table Creates a table and contains all table elements.
<thead> Table Head Declares the headers/titles of the table.
<tbody> Table Body Contains the body of the table
<tr> Table row Creates a row in the table and contains the data within the row.
<td> Table Data Contains the data within the table.
<tfoot> Table Foot Creates the row in the footer of the table.

Table Attributes

Attribute Name Description
colspan Column Span Defines the number of columns td will fill.
rowspan Row Span Defines the number of rows a td will fill.
